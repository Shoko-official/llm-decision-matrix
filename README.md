# LLM Decision Matrix

`llm-decision-matrix` defines the decision matrix workspace for the Modern LLM
Systems 2026 / arXiv Report program.

This repository will organize decision criteria and tradeoff tables after the
architecture taxonomy foundations exist.

It is not the research ledger, paper repository, architecture taxonomy source,
RAG implementation, inference benchmark, agent runtime, or memory layer.

## Repository Role

This repository owns:

* decision matrix governance;
* future decision criteria structure;
* tradeoff categories once approved;
* validation for matrix files once introduced;
* paper-facing decision support when backed by approved evidence.

The central project board is:

* [Modern LLM Systems 2026 / arXiv Report](https://github.com/users/Shoko-official/projects/4)

## Current Scope

Milestone 0 is limited to governance.

Included:

* repository scope;
* roadmap;
* contribution rules;
* review rules.

Out of scope:

* decision matrix content;
* scoring models;
* scientific claims;
* diagrams or figures;
* schema design.

## Evidence Policy

Future matrix criteria must reference approved research ledger or taxonomy
material, or stay clearly marked as unresolved planning notes.

Unsupported claims must not be used as paper-ready matrix content.
