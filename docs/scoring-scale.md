# Scoring Scale

This document defines the strict rubric and meaning for decision matrix score values (0-3).

## Scoring Scale Table

| Score | Meaning | Rubric Description |
|---|---|---|
| 0 | None / Not Applicable | The criterion is not met or there is no supporting evidence. |
| 1 | Low / Limited | The criterion is partially met with limited evidence. |
| 2 | Medium / Sufficient | The criterion is mostly met with sufficient evidence. |
| 3 | High / Full | The criterion is fully met with complete evidence. |

## Current Limits

Do not use scores outside the 0-3 range. All scores must be backed by evidence references.
